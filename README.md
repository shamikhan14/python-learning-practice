# Python Learning Practice

This repository contains my Python practice programs.

## Topics Covered

- Variables
- If else
- For loop
- While loop
- Break and continue
- Lists
- Dictionaries
- Functions
- Input
- Basic real-world programs

## Programs

### login_system.py
A simple login system with 3 attempts.  
If username and password are correct, login is successful.  
If all 3 attempts fail, the account is locked.

### claims_summary.py
A claims summary program using a list of dictionaries.  
It calculates pending claim count and pending claim total amount.
